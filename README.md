# prec - Primitive recursion interpreter 

## Dependencies/Compilation

No dependencies are required besides Haskell itself.
To compile, simply run `make`.

## Usage

```
% ./prec
Prec> Prec[S o P 2 3, P 1 1]
Enter list of 2 numbers: [1,2]
[3]
```
